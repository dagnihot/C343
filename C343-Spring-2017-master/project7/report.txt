English description:

For this project, I decided to implement the Breadth First Search algorithm.
BFS traverses through a tree or graph. It starts at the root of the tree and
explores the neighboring nodes before moving onto next level neighbors. BFS 
differs from depth first search algorithm because it uses a queue instead of
a stack.



Pseudocode for BFS:

Breadth-First-Search(Graph, root):
    
    create empty set S
    create empty queue Q      

    add root to S
    Q.enqueue(root)                      

    while Q is not empty:
        current = Q.dequeue()
        if current is the goal:
            return current
        for each node n that is adjacent to current:
            if n is not in S:
                add n to S
                n.parent = current
                Q.enqueue(n)
